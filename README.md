# Avoid Virus Game

## 1. 개요

시스템 프로그래밍 5팀의 프로젝트입니다.   
수업을 통해 배웠던 리눅스의 개념들을 활용하여   
바이러스를 피하는 콘솔 게임을 제작하였습니다.

## 2. 컴파일
```
$ cc main.c -lncurses -o main
```
## 3. 게임화면 & 게임방법

 * 게임화면은 총 3개로 구성되어있고, 키보드 방향키 <- , -> 를 활용하여 플레이어를 좌우로 움직여 바이러스를 피하면 됩니다.
